# Brick Capes – SVG Library

This repository contains **SVG files for fabric capes designed for brick figures**. The SVGs are intended for **laser cutting, Cricut, Silhouette, and other digital cutting methods**.

The goal is to make it easy to create custom fabric capes with clean edges and consistent sizing.

---

## What Goes in This Repo

- SVG files representing **cape cut‑out shapes**
- Files should be **1:1 scale**, ready to cut
- Designs may be:
  - Scanned from real capes
  - Digitally recreated
  - Original designs

Please keep files:
- Simple (single outline paths preferred)
- Clean (no unnecessary points)
- Named descriptively

Example: 
classic-short-cape.svg
flowing-long-cape.svg
hooded-cape.svg

---

## How to Contribute

1. **Create or scan a cape**
   - Scan a physical cape flat, or
   - Design one digitally (Inkscape, Illustrator, Affinity, etc.)

2. **Convert to SVG**
   - Use a single closed path for the cut line
   - Remove fills, strokes not required for cutting

3. **Verify scale**
   - Import the SVG into your cutter software
   - Confirm it matches real minifigure dimensions

4. **Submit**
   - Upload the SVG via GitHub
   - Include notes if needed (fabric tested, scaling notes, etc.)

## Using the SVGs

### Glowforge (Laser Cutting)

1. Upload the SVG into the Glowforge web interface
2. Set the outline as a **cut operation**
3. Use **proofgrade-equivalent or manual fabric settings**
4. Test on scrap before final cuts

⚠️ **Do NOT laser cut unsafe fabrics** (see safety section below).

---

### Cricut / Silhouette / Hobby Cutters

1. Upload SVG into Design Space or Silhouette Studio
2. Ensure the design imports at **100% scale**
3. Assign the path as a **cut**
4. Place fabric on a **fabric mat** or bonded to a stabilizer
5. Use a **rotary blade** where available

---

## Fabric Stiffening & Finishing

Light fabric stiffening helps capes hold their shape:

- **Matte medium + water** (commonly ~1:1 to 1:2 ratio)
- Brush on lightly
- Let dry fully before cutting or after shaping

You can tint the mixture with:
- Acrylic paint
- Fabric paint
- Ink (test first)

This helps reduce fraying and gives the cape a more “heroic” drape.

---

## ⚠️ Safety: Fabric & Laser Cutting

**Not all fabrics are safe to laser cut.**

🚫 **Never laser cut fabrics containing:**
- PVC / Vinyl
- Faux leather
- PU coatings
- Unknown synthetic blends

✅ Commonly safer options (still verify):
- 100% cotton
- Linen
- Wool (with ventilation)

When in doubt:
- Use a blade-based cutter (Cricut / Silhouette)
- Or check material safety documentation before laser use
